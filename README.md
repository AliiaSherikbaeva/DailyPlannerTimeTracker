
# 🗓️ Daily Planner with Time Tracker

A web application that allows users to plan their day and track how much time is spent on different tasks.

---

## 📌 Features

- Add and manage daily tasks
- Start/stop timers to track time spent
- View summary of time usage
- Responsive web UI

---

## 🛠️ Tech Stack

| Layer     | Technology     |
|-----------|----------------|
| Frontend  | React.js       |
| Backend   | Node.js + Express |
| Database  | MongoDB / PostgreSQL (choose one) |
| Version Control | Git + GitHub |
